# Madlibs Generator Project

This is a simple Python project that generates a fun story using user inputs.  
The program reads a story from a text file and replaces words like noun, adjective, and verb with user-provided values.

---

## Features
- Takes user input (noun, verb, adjective, etc.)
- Replaces placeholders in the story
- Uses an external `story.txt` file
- Interactive and beginner-friendly Python project

---

## Files in This Project
- `Madlibs_Generator.py` - Main Python program
- `story.txt` - Story template with placeholders
- `README.md` - Project description

---

## How to Run
1. Download all files (`Madlibs_Generator.py` and `story.txt`)
2. Open terminal or VS Code in the project folder
3. Run the command:

```bash
python Madlibs_Generator.py
